# Handwritten Character Recognition

## Overview

This project focuses on **recognizing handwritten characters** using machine learning and deep learning techniques. The model learns from labeled handwritten character images and predicts the corresponding character.

## Objective

* Preprocess handwritten image data
* Train a classification model for character recognition
* Evaluate model performance using standard metrics
